# SOW Mini App

## Project Structure
- `/client`: Vite (React) frontend.
- `/server`: Node.js Express backend.
- `/database`: Schema for PostgreSQL.

## Instructions
1. Setup PostgreSQL and execute `/database/schema.sql`.
2. Configure `.env` in the `/server` folder.
3. Run `npm install` in both `/client` and `/server`.
4. Run `npm run dev` in both directories to start the development environment.
